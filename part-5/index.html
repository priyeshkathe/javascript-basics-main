Old Way vs New Way Comparison

String Concatenation

Old Way

'Hello ' + name + ', you are ' + age

New Way (Template Literals)

`Hello ${name}, you are ${age}`

Function Syntax

Old Way

function(x) { return x * 2; }

New Way (Arrow Function)

x => x * 2

Extracting Properties

Old Way

var name = user.name;
var age = user.age;

New Way (Destructuring)

const { name, age } = user;
