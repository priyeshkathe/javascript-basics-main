## ğŸš€ Getting Started

### Prerequisites
- Basic HTML/CSS knowledge
- A modern web browser (Chrome, Firefox, Edge)
- A code editor (VS Code recommended)

### How to Use This Repository

1. **Clone or download** this repository
2. **Start with Part 1** - read the README, then open `index.html` in your browser
3. **Complete exercises** in `script.js` for each part
4. **Progress sequentially** - each part builds on the previous
5. **Finish with Homework** - the final project tests everything

### Running the Code
1. Open any `index.html` file directly in your browser
2. Open Developer Tools (F12) to see console output
3. Edit `script.js` and refresh the browser to see changes
